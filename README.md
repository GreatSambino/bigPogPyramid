# bigPogPyramid
Silly little project to test automated input and GUI in python.

Automatically types emote "pyramids" for Twitch chat. Also works with messaging apps like Discord.

Prebuilt executables:
- Google Drive (includes single file .exe): https://drive.google.com/drive/folders/1Hy4TYweAODtSokhIQkM6M0T-F5v0TUbN?usp=sharing
- Itch.io (distributable only): https://sambinodev.itch.io/bigpogpyramid
